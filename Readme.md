Vue Native Github Finder

## Introduction

Fins last 50 projects of a Github user  with profile and links

For Android/IOS

## Requirements

NPM/Node
 Yarn
Vue Native cli
Expo cli

## Installation

Clone and run yarn

## Configuration

npm start

Then various options with Expo to preview
